Key Concepts Covered
- Tokens: atomic units of model input/output.
- Context window: maximum tokens processed in a single request.
- Temperature: controls response randomness (0 = deterministic, higher = more creative).
- System prompt: instructions that define assistant behavior.
- LangChain chains: composable pipelines for LLM calls and processing.
- Streaming: real-time token-by-token output for responsive UX.

Security & Next Steps
- Do not commit secrets. Keep them in `.env` (ignored) and use a proper secret manager for production.
- If `.env` or any secret was committed previously, remove it from history using BFG or `git filter-repo` and rotate compromised keys.
- Consider adding a GitHub Actions secret store and documentation for key rotation.
